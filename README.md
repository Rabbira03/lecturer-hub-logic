# Lecturer Hub Logic

## Project Info
This is the Lecturer Module for the SWE 4070 Group Project 2.

## Technologies
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```

2. Start the development server:
   ```sh
   npm run dev
   ```
